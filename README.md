# Proyecto Node.js API en Vercel

Este proyecto es una API sencilla desplegada en Vercel utilizando Node.js.

## Endpoints disponibles

- **GET /api/products**  
  Devuelve un listado de productos de ejemplo en formato JSON.

## Despliegue

La API está disponible en:  
[https://proyecto-node-five.vercel.app/api/products](https://proyecto-node-five.vercel.app/api/products)
